# Operator runbook — AI founder mode

**Owner:** human (silent)  
**Operator:** AI agent (full execution)  
**Capital:** $0  
**Rule:** Never assign tasks to the owner unless a hard gate requires their identity/money rails.

## Hard gates (only times we need the owner)

1. **Receive money** — Stripe / Venmo / Cash App / bank under their name  
2. **Legal** — business registration, taxes, contracts in their name  
3. **Paid accounts** — anything that needs a card (avoid until profit)

Until first sale: no owner involvement.

## What the operator owns daily

- Ship product (sites, demos, templates)
- Maintain public sales site + portfolio
- Research prospects from public data
- Write personalized outreach packages
- Fulfill any inbound lead end-to-end
- Update scoreboard, pipeline, case studies
- Raise quality of demos until conversion is obvious

## Lead pipeline (no human required)

```
Public site form
  → GitHub Issue in spark-rabbit repo (title: [LEAD])
  → Operator triages same day
  → Free mockup built in /demos or /clients
  → Reply with live preview link
  → Close: 50% deposit instructions (owner payment link when ready)
  → Deliver → case study → ask referral
```

| Step | SLA |
